================================================================================
快速开始
================================================================================

1. 安装依赖
   uv sync

2. 验证系统
   python test_setup.py

3. 配置环境
   cp .env.example .env
   export OPENAI_API_KEY=sk-...

4. 运行诊断
   python main.py

5. 启动 API
   python -m uvicorn app.api:app --reload
